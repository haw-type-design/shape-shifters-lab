You might ask yourself: »What on earth is a variable font and how do I get started with it?«. Bear with me. In the course of this semester we will get the hang of it and fully understand the whats, hows and abouts of this technology. Please don’t be shy, ask any question that comes to your mind and pull the most out of this course. But let’s start right at the beginning and get right to the core: What is a variable font?
Really simplifying things, one could say: »it’s many fonts in one«. This was – back in the early days of variable fonts – quite a burden, especially on the web. Back then
the glyph shapes in modern fonts [were] restricted to a single, static configuration. Any variation in weight, width, stroke contrast, etc.—no matter how subtle—require[d] separate font files.
1
Nick Sherman: Variable Fonts for Responsive Design, 2015 link
Since then variable fonts have seen increasing use on the web – but not only in this medium. However, many uses are still restricted to the »classic« styles of typography as are the above mentioned variables – i.e. weight, width etc.

Speaking of variables: for now we only slightly touched what variable fonts are and how they work. So let us dive into some terminology.
Variable fonts can be varied along so called axes. This just means, that each variable – that’s also where the name variable fonts comes from – such as weight, width, you name it, is attached to an axe along which the font can be interpolated. This again means, that each point that describes the outline of each glyph will be translated between two or more points – the so called drawn instances. Drawn instances are two or more actual outline drawings of letter shapes. Preferably they sit at the extremes of each axis. So in the example of a weight axis this would be an Extra Light and an Extra Bold weight for example. Axes can be separate from one another or interdependent. This means that for example you can have a weight axis and another axis for an italic angle. If everything should work together, so for example you want a Light Semi-Italic, you will create something called a Designspace. You can learn more about it in this quick video here.
